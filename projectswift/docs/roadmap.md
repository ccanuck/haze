# Roadmap

## Near-term

- complete gateway websocket transport implementation
- implement full command schema sync to Discord
- add autocomplete request/response path
- add collectors for buttons, selects, and modals

## Mid-term

- manager-style APIs for guilds/channels/users/messages
- richer cache invalidation and partial object handling
- multipart upload support for attachments
- role/permission utility layer

## Long-term

- voice and stage support
- horizontal scaling helpers for large shard fleets
- deeper plugin runtime hooks
- official adapter packages and observability integrations
