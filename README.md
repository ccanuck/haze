# haze
Haze is a modern swift-native Discord framework made to make bot development in swift fast, efficient, and enjoyable. Haze is one of the most comprehensive swift-native Discord frameworks, Haze provides a clean api for interactions, commands, components, events and gateway handling, all built on swift.
